# nexe-webpack-sample
nexe most simple sample with webpack

Please let me know if there is a better way.

## Feature
* console output sample
* create bundle.js with webpack 
* create windows x64 executeble binary with nexe

## Development
1. init
```sh
$ npm install
```

2. build
```sh
$ npm run build
```
